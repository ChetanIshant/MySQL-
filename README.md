## Purpose of This Repository

The goal of this repository is to:
- Strengthen core SQL concepts
- Apply SQL to **real-world data scenarios**
- Build a solid foundation for **software development, QA, and data-related roles**

## Topics Covered

This repository includes practice and examples related to:

- Basic SQL queries (`SELECT`, `WHERE`)
- Filtering and sorting data
- Aggregate functions (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- `GROUP BY` and `HAVING`
- Joins (`INNER`, `LEFT`, `RIGHT`)
- Subqueries
- Data manipulation (`INSERT`, `UPDATE`, `DELETE`)
- Table structure changes (`ALTER`, `DROP`)
- Basic real-world query scenarios
- Debugging and fixing SQL errors
